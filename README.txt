DEPENDENCES/MODULES A INSTALLER POUR FAIRE FONCTIONNER L'API
- Angular
- EXPRESS (le serveur)
- NODEMON (pour simplifier le lancement du serveur node)
- MONGOOSE (pour se connecter à la bdd MongoDB)
- JSONWEBTOKEN (pour utiliser un token d'authentification)
- DOTENV (variables d'environnement pour sécuriser des données tel quel le port du serveur ou la clé du token d'authentification)
- BCRYPT (hash du mot de passe)

-npm start sur le frontend pour lancer l'application angular
-nodemon server sur le backend pour lancer le serveur
